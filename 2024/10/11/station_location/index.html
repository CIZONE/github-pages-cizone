水文站点经纬度获取方式

Cizony Lv1
  2024-10-11 15:17:57 2024-10-11 15:17 创建   2024-10-11 15:50:07 2024-10-11 15:50 更新

全国水雨情信息网 mwr.cn

该网站记录了各水文站点的经纬度，获取流程为：

  • F12打开控制台
  • 上方选项卡选择网络
  • 在网站的地图查询窗口中搜索需要的水文站点名称
  • 左侧名称栏选择mapsearch，可能有几个空的，找到有信息的那个
  • stnm就是站点名称，找到对应站点名称的lgtd和lttd，即为经纬度

Base64二次解码

上述获取的经纬度是字母+数字+等号的形式，是经纬度的Base64二次编码形态，需要进行两次解码获取常见数字形态的经纬度，二次解码的python代码如下：

import base64

def decode_base64(encoded_data):
    # 解码Base64
    decoded_data = base64.b64decode(encoded_data).decode('utf-8')
    decode2 = base64.b64decode(decoded_data).decode('utf-8')
    return decode2

def decode_base64_batch(encoded_data_list):
    decoded_results = []
    for encoded_data in encoded_data_list:
        decoded_results.append(decode_base64(encoded_data))
    return decoded_results

# 输入的Base64编码字符串列表
encoded_data_list = [
    # 把获取到的base64二次编码的经纬度替换到这里
    # 可以批量，一行行堆叠即可
    # 例：
    "TXpBdU5ERTJOalky",
    "TXpBdU5BPT0=",
    "TVRFeExqRTJOalkyTmc9PQ==",
    "TXpBdU5EWTNOemM0",
]

# 调用函数并打印结果
results = decode_base64_batch(encoded_data_list)
for i, result in enumerate(results):
    print(f"{result}")
